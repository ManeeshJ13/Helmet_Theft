<h1>AI-Driven IoT Framework for Helmet Theft Prevention Using Sensor Fusion and Behavioral Anomaly Detection</h1>

<h3>OVERVIEW</h3>
Helmet theft is a common problem for two‑wheeler riders. This project addresses the issue by embedding sensors and a microcontroller inside the helmet to monitor motion. When suspicious movement is detected, the system sends a real‑time alert to the user.

 <h3>FEATURES</h3>
 <ul>
   <li>Motion detection using Accelerometer & Gyroscope (MPU6050)</li>
   <li>Real‑time SMS alerts via SIM800L GSM module </li>
   <li>User‑controlled enable/disable functionality </li>
   <li>Compact, low‑cost, and helmet‑mountable design </li>
   <li>Scalable for future upgrades </li>
 </ul>

<h3>HARDWARE COMPONENTS</h3>
<ul>
  <li>ESP32 Microcontroller</li>
  <li>MPU6050 - Gyroscope and Accelerometer</li>
  <li>SIM800L GSM Module</li>
  <li>Power Supply and Battery</li>
  <li>Smartphone </li>
</ul>
